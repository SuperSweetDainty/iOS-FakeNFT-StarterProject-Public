# Декомпозиция эпика Каталог



## Модуль 1:

#### Верстка
- Верстка основного экрана CatalogViewController (UITableView) (est: 3.5 часа; fact: 2.5 часов).
- Верстка ячейки с обложкой, названием и количеством NFT (est: 3 часа; fact: 2 часов).
- Верстка loading state и empty state для таблицы (est: 1.5 часа; fact: 1 час).
- Верстка navigation bar с кнопкой сортировки (est: 1 час; fact: 1 час).


#### Логика
-  Сортировка таблицы по названию, количеству NFT (est: 3 часа; fact: 3 часа).
-  Сохранение выбранного способа сортировки (UserDefaults)(est: 2 часа; fact: 2 часа).
-  Навигация с CatalogViewController на CatalogCollectionViewController  (est: 1.5 часа; fact: 1.5 часов).
-  Обработка ошибок и "Повторить" для каталога (est: 1 час; fact: 1 часов).

#### Логика
- Загрузка деталий коллекции (est: 2 часа; fact: 1 час).
- Реализация логики добавления/удаления из избранного (est: 2 часа; fact: 2 часа).
- Реализация логики добавления/удаления из корзины (est: 2 часа; fact: 1 час).
- Открытие сайта автора (WKWebView) (est: 2 часа; fact: 2 часа).
- Обработка состояний загрузки и ошибок для коллекции (est: 1.5 часа; fact: 1 час).
- Обновление UI состояния кнопок (лайк/корзина) при изменениях (est: 1 час; fact: 1 час).
- Empty state для пустой коллекции (est: 1 час; fact: 1 часов).
- Интеграция со стореджами для обновления UI при изменениях (est: 2 часа; fact: 2 часа).

- "Повторить" для ошибок загрузки коллекции (est: 0.5 часа; fact: 0.5 часов).



## Модуль 3:

- Создать GET запрос для получения данных NFT по IDs (est: 1.5 часа; fact: 1.5 часа).
